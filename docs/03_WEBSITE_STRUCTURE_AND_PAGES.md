# NCLEX Prep Nation — Full Website Structure and Page Specification

## 1. Document Purpose

This document defines the complete website structure for NCLEX Prep Nation.

It tells the AI developer what pages to build, what each page should contain, how pages should connect, what CTAs to use, and how the website should support public visitors, students, admins, SEO, and manual package requests.

The website must be built as a complete preparation platform, not a simple landing page.

---

## 2. Website Structure Overview

NCLEX Prep Nation will include:

1. Public marketing pages
2. NCLEX exam preparation pages
3. Practice question pages
4. Free diagnostic test system
5. Student dashboard
6. Resource library
7. Package/request access system
8. Manual WhatsApp/email booking flow
9. Admin dashboard
10. Blog/SEO system
11. Legal/trust pages
12. Contact/support pages

## 5. Homepage Specification

URL: `/`

The homepage is the main trust and conversion page.

### 5.1 Homepage Goals

The homepage must:

1. Explain what NCLEX Prep Nation does.
2. Build trust immediately.
3. Encourage users to start a free diagnostic test.
4. Show free and paid resources.
5. Present packages clearly.
6. Show the manual WhatsApp/email request system professionally.
7. Lead users to create an account or request access.

### 5.2 Homepage Hero Section

Headline:

> Prepare Smarter for NCLEX-RN and NCLEX-PN

Subheading:

> Practice with original NCLEX-style questions, NGN case studies, detailed rationales, study plans, mock exams, and guided support built to strengthen your clinical judgment.

Primary CTA:

> Start Free Diagnostic

Secondary CTA:

> View Prep Packages

Trust note:

> Independent NCLEX preparation platform. Original practice materials. No leaked exam questions.

Visual idea:

- Dashboard preview
- Question card preview
- Student progress preview
- Calm nursing student image

### 5.3 Trust Strip

Show trust badges:

- Original Practice Questions
- Detailed Rationales
- NGN-Style Case Studies
- NCLEX-RN & NCLEX-PN Prep
- Student Progress Tracking
- WhatsApp Support Available

### 5.4 Problem Section

Headline:

> NCLEX preparation can feel overwhelming without a clear plan.

Content should mention:

- Too many resources
- No clear study plan
- Weak clinical judgment
- Difficulty with NGN questions
- Poor progress tracking
- Fear of failure

### 5.5 Solution Section

Headline:

> NCLEX Prep Nation gives you a structured path.

Steps:

1. Take a free diagnostic test.
2. Identify weak areas.
3. Practice by topic.
4. Review detailed rationales.
5. Track your progress.
6. Request guided support when needed.

### 5.6 Free Diagnostic Section

Headline:

> Start with a free diagnostic test.

Content:

> Answer a short set of NCLEX-style questions and receive feedback on your preparation level, strong areas, and weak topics.

## 10. Free Diagnostic Test Page

URL: `/free-diagnostic`

### Goal

Convert visitors into leads and recommend packages.

### Sections

1. What the diagnostic test is
2. What students will receive
3. Topics covered
4. Estimated completion time
5. Start button
6. Trust note

CTA:

> Start My Free Diagnostic

### Diagnostic Flow

1. User opens diagnostic page.
2. User clicks start.
3. User answers a set of questions.
4. User submits test.
5. System calculates score.
6. System shows partial result.
7. User enters email/WhatsApp to unlock full report.
8. System displays full result.
9. System recommends resources or package.

### Result Page Should Show

- Overall score
- Strong areas
- Weak areas
- Recommended package
- Recommended next study action
- CTA to request access
- CTA to contact WhatsApp

## 32. Conversion Path

The main conversion path:

1. User lands on homepage or blog.
2. User clicks free diagnostic.
3. User completes diagnostic.
4. User enters email/WhatsApp.
5. User sees weak areas.
6. User receives package recommendation.
7. User clicks request access.
8. Admin contacts user.
9. User pays manually.
10. Admin grants access.

The website should support this path clearly.

## 34. Final Website Structure Summary

NCLEX Prep Nation should be structured as a complete learning and exam preparation platform.

The site should include:

- Public pages for trust and SEO
- Practice tools for engagement
- Student dashboard for retention
- Admin dashboard for management
- Manual package request flow for sales
- Legal pages for legitimacy
- Blog pages for traffic

The manual payment system should feel like a personalized access request process, not an incomplete checkout system.
