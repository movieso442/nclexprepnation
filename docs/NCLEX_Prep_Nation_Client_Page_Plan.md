# NCLEX Prep Nation Website Page Plan

**Short client approval document before development**

## Project Direction

The website will be built as a clean and trustworthy NCLEX preparation platform called NCLEX Prep Nation. The main goal is to help students prepare for NCLEX-RN and NCLEX-PN with practice questions, study resources, mock exams, and support.

The site will not look like a random question website. It will feel like a real learning platform where a student can start free, test their level, request materials, and later continue learning from a student dashboard.

Since online payment API is not part of the first stage, the package process will be handled through WhatsApp and email. This will be presented as a guided request process, not as a missing payment system. Students will request access, then the team will contact them with the next steps.

## Main Website Pages and Content

**1. Home Page** — The home page will introduce NCLEX Prep Nation clearly. It will show what the site does, who it is for, and why students should trust it. It will include a strong message, a free diagnostic button, sample question preview, resource preview, package preview, testimonials, FAQ, and WhatsApp contact option.

**2. About Page** — This page will explain the purpose of NCLEX Prep Nation, the type of students it supports, and the kind of resources it provides. It will also make it clear that the platform is independent and provides original preparation content.

**3. NCLEX-RN Page** — This page will focus on students preparing for NCLEX-RN. It will include RN practice areas, study resources, question categories, NGN support, and a call to request NCLEX-RN materials or start a diagnostic test.

**4. NCLEX-PN Page** — This page will focus on students preparing for NCLEX-PN. It will explain the available PN preparation materials, practice questions, topic support, and package options for PN candidates.

**5. Practice Questions Page** — This page will show the question bank structure. Students will see free sample questions, question categories, question types, and a preview of detailed rationales. Some questions will be free, while full access will require a package request.

**6. Free Diagnostic Test Page** — This page will allow students to answer a short set of NCLEX-style questions. After the test, the student will see their score, weak areas, strong areas, and a recommended package or study direction.

**7. Study Resources Page** — This page will contain study guides, checklists, topic summaries, pharmacology notes, lab values, prioritization guides, infection control guides, and exam day preparation materials. Some resources will be free, while premium resources will be locked.

**8. Packages Page** — This page will explain the Free, Starter, Premium, and Guided Support packages. The buttons will use simple wording like Request Access, Book Materials, Get Study Materials, and Contact Us on WhatsApp.

**9. Request Access / Book Materials Page** — This page will collect the student’s name, email, WhatsApp number, country, exam type, preferred package, and message. After submission, the client team will contact the student by WhatsApp or email.

**10. Blog Page** — The blog will help bring students from Google and social media. It will include articles on NCLEX study plans, NGN questions, pharmacology, prioritization, SATA questions, mock exams, and exam day tips.

**11. Contact and FAQ Pages** — The contact page will give students a simple way to reach the team. The FAQ page will answer common questions about packages, materials, WhatsApp booking, NCLEX-RN, NCLEX-PN, and platform access.

## Student Side of the Website

A student dashboard will be included so the website feels complete and useful after signup. The dashboard will show the student’s package status, diagnostic result, weak topics, practice progress, saved questions, mock exam results, and recommended next action.

Students will be able to continue practice, review rationales, save questions, download allowed resources, and request an upgrade when they need more access. This will help keep students coming back instead of visiting once and leaving.

## Admin Side of the Website

The admin dashboard will help manage the business side of the platform. The admin will see students, leads, package requests, manual payment status, access status, contact messages, questions, resources, blog posts, and testimonials.

Because payment will be handled manually at the beginning, the admin dashboard will be very important. The admin will be able to mark a student as contacted, waiting for payment, payment confirmed, access granted, or materials sent. This will keep the process organized.

## Trust and Legitimacy

The website will be clear that NCLEX Prep Nation is an independent preparation platform. It will not claim to be the official NCLEX website and it will not use official logos without permission.

The site will also state that it provides original practice questions and study resources. It will not claim to provide leaked or real exam questions. This is important because the site must look professional, ethical, and safe for students.

## Approval Note

Once this page plan is approved, I will use it as the base for development. The website will be built around trust, useful content, student retention, and simple package requests through WhatsApp and email.

The final result will be a full NCLEX preparation website, not just a landing page. It will have public pages for visitors, a student dashboard for learners, and an admin dashboard for managing the business.
